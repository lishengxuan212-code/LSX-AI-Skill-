# xy-ai-ui

A lightweight Codex skill for consistent desktop UI/UX and task-flow design.

## What it enforces
- Unified visual language and spacing
- Symmetric layout and scrollbar-safe alignment
- Safe destructive flows: confirm -> execute -> cancel -> result
- Clear Main / Bridge / Renderer separation for Electron apps

## Primary use cases
- UI consistency refactors
- Modal/progress/cancel interaction redesign
- Long-running task UX and state-machine cleanup
- Building concise UI/interaction standards for teams
